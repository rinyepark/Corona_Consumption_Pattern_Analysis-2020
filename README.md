## 1. Duration
- 2020.10 ~ 2020.12

## 2. Skills
- Python, SQL, k-means Clustering
- Jupyter notebook, Google BigQuery

## 3. Contents
- 분석배경  
  코로나 바이러스를 예방하기 위해 사회적 거리두기를 시행하며 변화하는 생활패턴/소비양상을 분석해 포스트 코로나시대 대비
- 분석내용
  - 코로나로 인한 소비자들의 생활 및 소비 변화 파악
  - 포스트코로나 시대에 대비해 활용가능 인사이트 도출
- 사용 데이터 및 분석환경
